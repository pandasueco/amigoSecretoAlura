# amigoSecretoAlura

Este é um projeto experimental, em formato de exercício, parte do curso "Iniciante em Programação G8 - ONE" da Alura em parceria com a Oracle.

Basicamente, trata-se de um site que sorteia nomes para um Amigo Secreto.
Coloca-se os nomes e, aleatoriamente, o site sorteia quem fica com quem.
